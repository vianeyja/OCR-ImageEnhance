# OCR-ImageEnhance
> Optimización de imágenes para el API de "Imagen a Texto" (OCR) de [Computer Vision](https://azure.microsoft.com/en-us/services/cognitive-services/computer-vision/)

Este documento presenta cómo generar una función de Azure de tipo [HTTP Trigger](https://docs.microsoft.com/es-es/azure/azure-functions/functions-triggers-bindings) que recibe la url de una imagen, y la almacena en un blob storage en Azure en escala de grises y con contraste añadido.
